# HC05 Bluetooth
To create bluetooth connection, run the following commands from the RPi terminal:

**Connect RFCOMM**
`sudo rfcomm connect hci0 98:D3:32:20:82:9D 1`

**Release RFCOMM**
`sudo rfcomm release 0`
